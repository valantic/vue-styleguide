# Blueprint: Playwright Test Agents Setup (Planner/Generator/Healer) + Seed E2E

## Summary
Set up Playwright test infrastructure in this Vue/Vite repo with a minimal, stable E2E seed test and Playwright Agents scaffolding, while keeping existing unit-test tooling intact.

Chosen decisions:
- Agent loop: `opencode`
- E2E folder: `tests/e2e`
- Optional test IDs: include now

## Scope Split

### Minimal
- Install Playwright Test and browsers
- Add Playwright config + E2E scripts
- Initialize Playwright Agents and commit generated artifacts
- Add/update seed spec for app-ready assertion
- Add developer docs for local run + agent workflow

### Optional (included in this implementation)
- Add stable `data-testid` hooks in app shell/layout

### Optional (not included in this implementation)
- CI workflow for E2E jobs

## Files and Directories

### Create
- `docs/blueprints/playwright-agents-setup.md`
- `docs/playwright-agents.md`
- `playwright.config.ts`
- `tests/e2e/seed.spec.ts`
- `specs/` (generated by `init-agents`)
- `.opencode/**` and `opencode.json` Playwright Agent definitions (generated by `init-agents --loop=opencode`)
- `.github/playwright-agents/**` mirrored Playwright Agent prompts for repository visibility

### Change
- `package.json`
- `package-lock.json`
- `src/styleguide/styleguide.vue`

## Configuration Values
- E2E env var: `E2E_BASE_URL`
- Default local URL: `http://127.0.0.1:4173`
- Seed route: `/sg/test-readme`
- Playwright `webServer`:
  - `command`: `npm run dev -- --host 127.0.0.1 --port 4173 --strictPort`
  - `url`: `http://127.0.0.1:4173`
  - `reuseExistingServer`: `!process.env.CI`
  - `timeout`: `120000`
- Behavior rule: if `E2E_BASE_URL` points to a non-local target, do not auto-start the local server.

## Command Sequence
1. `npm install -D @playwright/test`
2. `npx playwright install`
3. `npx playwright init-agents --loop=opencode`
   - Fallback: `npx playwright@latest init-agents --loop=opencode`
4. `npm run test:e2e`
5. Optional local UI run: `npm run test:e2e:ui`

## Minimal Seed Test
- Location: `tests/e2e/seed.spec.ts`
- Behavior:
  - Navigate to `/sg/test-readme` (via `baseURL`)
  - Assert `data-testid="app-shell"` is visible
  - Assert `data-testid="page-content"` is visible
  - Keep comment placeholder for future auth `storageState` setup

## Validation Checklist (Acceptance Mapping)
- Fresh clone and dependency install complete without errors
- `npx playwright test` (or `npm run test:e2e`) passes seed test
- Agent definitions exist under `.opencode/**` and mirrored definitions exist under `.github/playwright-agents/**`
- `specs/` directory exists with starter content
- `docs/playwright-agents.md` documents setup and workflows clearly
- `data-testid` hooks exist in styleguide shell/layout
